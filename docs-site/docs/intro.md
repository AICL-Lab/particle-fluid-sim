# WebGPU Particle Fluid Simulation

A high-performance particle fluid simulation running entirely on the GPU using WebGPU compute shaders. Features 10,000+ particles with real-time physics, adaptive quality scaling, and a polished interactive demo.

## Quick Start

```bash
git clone https://github.com/LessUp/particle-fluid-sim.git
cd particle-fluid-sim
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in a WebGPU-compatible browser.

## Requirements

- Node.js 18+
- Chrome 113+, Edge 113+, or Safari 17+
