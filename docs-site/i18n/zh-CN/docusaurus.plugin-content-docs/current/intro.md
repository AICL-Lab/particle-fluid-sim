# WebGPU 粒子流体模拟

基于 WebGPU 计算着色器的高性能粒子流体模拟。支持 10,000+ 粒子的实时物理计算、自适应质量缩放，以及精美的交互式演示。

## 快速开始

```bash
git clone https://github.com/LessUp/particle-fluid-sim.git
cd particle-fluid-sim
npm install
npm run dev
```

在支持 WebGPU 的浏览器中打开 [http://localhost:5173](http://localhost:5173)。

## 系统要求

- Node.js 18+
- Chrome 113+、Edge 113+ 或 Safari 17+
