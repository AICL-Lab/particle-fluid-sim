# 架构

WebGPU 粒子流体模拟的系统架构概述。

## CPU-GPU 异构计算

```
┌─────────────────────────────────────────────────────┐
│  CPU (TypeScript)                                    │
│  • WebGPU 初始化                                     │
│  • 质量检测与缩放                                    │
│  • 输入处理（鼠标/触摸）                             │
│  • 帧循环编排                                        │
└───────────────────┬─────────────────────────────────┘
                    │ writeBuffer / commandEncoder
                    ▼
┌─────────────────────────────────────────────────────┐
│  GPU (WGSL 着色器)                                   │
│                                                       │
│  ┌─────────────┐  ┌──────────┐  ┌──────────────┐    │
│  │  1.计算     │─▶│ 2.轨迹   │─▶│ 3.渲染       │    │
│  │  物理着色器 │  │ 效果    │  │  粒子        │    │
│  └─────────────┘  └──────────┘  └──────┬───────┘    │
│                                         │            │
│                                         ▼            │
│                                ┌──────────────┐     │
│                                │ 4.呈现       │     │
│                                │ 到屏幕       │     │
│                                └──────────────┘     │
└─────────────────────────────────────────────────────┘
```

## 物理管线

每个粒子每帧执行以下更新步骤：

1. **应用重力** → `velocity += gravity * deltaTime`
2. **鼠标排斥** → 在半径内推开粒子
3. **限制速度** → 限制在 `MAX_SPEED`
4. **更新位置** → `position += velocity * deltaTime`
5. **边界反弹** → 带阻尼的弹性碰撞

## 关键模块

| 模块                    | 功能                       |
| ----------------------- | -------------------------- |
| `src/core/webgpu.ts`    | WebGPU 设备和上下文初始化  |
| `src/core/buffers.ts`   | GPU 缓冲区创建和管理       |
| `src/core/pipelines.ts` | 计算和渲染管线设置         |
| `src/core/renderer.ts`  | 帧编排和渲染循环           |
| `src/core/physics.ts`   | CPU 端物理计算（用于测试） |
| `src/core/color.ts`     | 速度到颜色的映射           |
| `src/core/input.ts`     | 鼠标和触摸事件处理         |
| `src/core/quality.ts`   | 自适应质量检测             |
| `src/config/sim.ts`     | 模拟常量                   |
| `src/shaders/*.wgsl`    | GPU 着色器程序             |

## 数据流

```
粒子缓冲区 (GPU)
       │
       ▼
┌─────────────────┐
│ 计算着色器      │ ← Uniform 缓冲区（画布大小、鼠标、时间增量）
│ (physics.wgsl)  │
└────────┬────────┘
         │ 更新后的粒子
         ▼
┌─────────────────┐
│ 渲染着色器      │ → 轨迹纹理
│ (render.wgsl)   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 呈现着色器      │ → 屏幕
│ (present.wgsl)  │
└─────────────────┘
```

## 性能特征

| 指标       | 值             | 备注                       |
| ---------- | -------------- | -------------------------- |
| 粒子数     | 2,500 - 10,000 | 根据设备自适应             |
| 粒子大小   | 16 字节        | 4 × float32 (x, y, vx, vy) |
| 计算工作组 | 64 线程        | 针对 GPU 架构优化          |
| 帧预算     | < 16ms         | 目标 60 FPS                |
