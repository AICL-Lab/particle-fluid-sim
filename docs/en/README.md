## Technical Overview

The WebGPU Particle Fluid Simulation is built with:

- **GPU Acceleration**: WebGPU compute shaders for physics
- **TypeScript**: Type-safe client code
- **WGSL**: WebGPU Shading Language for GPU kernels
- **Adaptive Quality**: Runtime particle scaling
