## 技术概览

WebGPU 粒子流体仿真项目采用以下技术构建：

- **GPU 加速**: WebGPU 计算着色器执行物理运算
- **TypeScript**: 类型安全的前端代码
- **WGSL**: WebGPU 着色语言
- **自适应质量**: 运行时粒子数量动态调整
